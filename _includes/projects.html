Flying Drum Machine

A project in collaboration with WaveDNA (a music technology company) and the University of Toronto Institute for Aerospace Studies. A touch-sensitive quadcopter that sends path information to a Linux machine running ROS–the Robot Operating System – and then sends information over a network to a macbook running Ableton Live 9 and Max. This information is translated to MIDI and fed into an Ableton Live drum rack. The performance is being visualized using WaveDNA Liquid Rhythm.
